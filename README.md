# Conversational-AI-for-Personalized-Debt-Management

<h1 align="center">Hi 👋, We're Rahini Raneru and Vaisakhi Das from Team Pottycoders</h1>
<h3 align="center">and we present Debtora - Your Smart Financial Advisor Bot.</h3>

- 🔭 Goal: **Debtora is an intelligent financial advisor chatbot that helps users manage debt, income, expenses, and savings goals. By analyzing user inputs, it provides actionable insights and personalized financial recommendations, making financial planning accessible to everyone.**

- Features **🧮 Tracks and analyzes income, debt, and expenses 🎯 Sets and manages financial goals 📊 Generates personalized financial advice 🤖 Smart conversational interface 🔐 Privacy-first design – your data stays with you**
